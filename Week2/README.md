# 2주차: 가스 및 연기 감지 시스템 구현

### 💡 학습 내용

- **MQ-2 가스 센서의 원리**
  - 이산화주석($SnO_2$) 소자가 가스 농도에 따라 저항이 변하는 특성 이해
  - 가스 감지 시 **Low(0)** 신호를 출력하는 디지털 출력(D0) 로직 파악

- **`gpiozero` 라이브러리 활용**
  - `DigitalInputDevice` (가스 센서용) 클래스 사용법
  - `OutputDevice` (능동 부저용) 클래스 사용법

- **센서 민감도 물리적 튜닝**
  - 가변저항(Potentiometer) 조절을 통한 기준 전압(Threshold) 설정 원리 습득

- **능동 부저(Active Buzzer) 제어**
  - 별도의 발진 회로 없이 전원 인가만으로 경보를 울리는 하드웨어 특성 확인


### 🔌 하드웨어 연결 (Pin Mapping)

| 부품 | 핀 종류 | 라즈베리 파이 핀 |
| :--- | :--- | :--- |
| **MQ-2 가스 센서** | DOUT | **GPIO 17** |
| **능동 부저** | DOUT (IN) | **GPIO 18** |
| **공통 전원** | VCC / GND | 5V / GND |


### ⚠️ 주의사항 (Troubleshooting)

- **센서 민감도 조절**
  - 가스라이터 테스트 시 반응이 없다면, 모듈 뒷면의 파란색 가변저항을 **반시계 방향**으로 돌려 민감도를 높여야 함. (약 2바퀴 정도 조절 권장)

- **로직 반전 확인**
  - MQ-2 센서는 가스 감지 시 `value`가 **0**이 됨. 코드 작성 시 `if gas.value == 0:` 조건을 사용해야 함.

- **부저 전원 및 소리**
  - 능동 부저는 VCC와 GND가 제대로 연결되어야 하며, `on()` 함수 호출 시 즉시 단일음이 발생함. 핀 배열이 반대로 되지 않도록 주의.
