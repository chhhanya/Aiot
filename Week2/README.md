# 2주차: 가스 및 연기 감지 시스템 구현

### 💡 학습 내용

- **컴포넌트별 요구 전압 확인**
  - MQ-2 센서(5V)와 능동 부저(3.3V)의 전원 사양 차이에 따른 개별 회로 구성 습득
- **MQ-2 가스 센서의 원리**
  - 이산화주석($SnO_2$) 소자가 가스 농도에 따라 저항이 변하는 특성 이해
  - 가스 감지 시 **Low(0)** 신호를 출력하는 디지털 출력(D0) 로직 파악
- **`gpiozero` 라이브러리 활용**
  - `DigitalInputDevice` 및 `OutputDevice` 클래스를 이용한 입출력 제어

### 🔌 하드웨어 연결 (Pin Mapping)

| 부품 | 핀 종류 | 라즈베리 파이 핀 | 요구 전압 |
| :--- | :--- | :--- | :--- |
| **MQ-2 가스 센서** | VCC / GND | **5V (Pin 2/4)** / GND | **5V** |
| **MQ-2 가스 센서** | DOUT | **GPIO 17** | - |
| **능동 부저** | VCC / GND | **3.3V (Pin 1)** / GND | **3.3V** |
| **능동 부저** | I/P (Signal) | **GPIO 18** | - |

### ⚠️ 주의사항 (Troubleshooting)

- **전원 분리 연결 (CRITICAL)**
  - MQ-2 센서는 데이터시트에 따라 **5V** 전원에, 능동 부저는 **3.3V** 전원에 각각 따로 연결. 전원 혼용 시 소자 손상 위험이 있음.
- **센서 민감도 조절**
  - 가스라이터 테스트 시 반응이 없다면, 모듈 뒷면의 파란색 가변저항을 **반시계 방향**으로 돌려 민감도를 높여야 함.
- **로직 반전 확인**
  - MQ-2 센서는 가스 감지 시 `value`가 **0**이 됨. 코드 작성 시 `if gas.value == 0:` 조건을 사용해야 함.
