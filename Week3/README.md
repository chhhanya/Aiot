# 3주차: 인체 감지 및 자동 촬영 시스템 구현

### 💡 학습 내용

- **PIR(Passive Infrared) 센서의 메커니즘**
  - 적외선 변화를 감지하는 수동형(Passive) 센서의 특성 및 파이로전기(Pyroelectric) 소자의 원리 이해
  - 프레넬 렌즈(Fresnel Lens)를 통한 감지 각도 및 거리 확장 기술 습득
- **`gpiozero` 및 `picamera2` 라이브러리 연동**
  - `MotionSensor` 클래스를 활용한 디지털 이벤트 트리거 구현
  - `Picamera2` 라이브러리를 이용한 하드웨어 가속 기반 이미지 캡처 프로세스 설계
- **타임스탬프 기반 데이터 관리**
  - `datetime` 모듈을 활용하여 현재 시간을 파일명으로 변환하고 고유한 데이터 이력을 관리하는 기법 습득

⚠️ 주의사항 (Troubleshooting)
초기 안정화 시간 (Warm-up Time)

PIR 센서는 전원 인가 후 주변 적외선 환경을 학습하기 위해 약 1~2분의 안정화 시간이 필요하며, 이 기간에는 오작동이 발생할 수 있음.

감지 대기 지연 (Recovery Time)

한 번의 감지 이벤트 발생 후 센서가 다시 준비 상태로 돌아가기까지 하드웨어적으로 약 2초 내외의 지연 시간이 존재함.

카메라 자원 관리

프로그램 종료 시 반드시 picam2.stop()을 호출하여 카메라 인터페이스를 해제해야 함. 해제되지 않을 경우 다음 실행 시 장치 점유 에러가 발생함.

신호 특성 (Active-High)

본 실습의 PIR 센서는 움직임 포착 시 value가 1이 되는 'Active-High' 로직을 따르므로 조건문 작성 시 주의가 필요함.
