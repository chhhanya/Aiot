# AIOT 강의 실습 저장소

이 저장소는 AIOT 강의 및 실습 세션에서 진행한 소스 코드를 아카이빙하기 위한 공간.
라즈베리 파이와 센서를 활용한 하드웨어 제어 및 데이터 처리 실습 내용을 포함함.


##  레포지토리 구조

```text
Aiot/
├── .git/
├── README.md          (전체 프로젝트 메인 설명)
├── Week1/             (1주차 폴더)
│   ├── led_test.py    (작성한 코드)
│   └── README.md      (1주차 실습 메모)
└── Week2/             (2주차 폴더)
    └── ...
```

## 실습 환경

**HARDWARE:** Rasberry Pi
**Language:** Python 3
